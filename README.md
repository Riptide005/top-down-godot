# Top Down Godot

This project is a small Godot learning sandbox focused on understanding how game engines work from the inside out.

I’m starting with Godot so I can build up practical experience with scene setup, scripting, movement, and game structure before moving on to a larger original project.

This version is a top-down shooter clone inspired by a game I previously built in Pygame and published. Rebuilding it in Godot is helping me compare approaches, learn new tools, and deepen my game development experience.

The long-term goal is to use what I learn here to build my own original dream game on the side.
